# InterviewAce 🚀

A Full Stack Interview Preparation Platform built using React.js, Spring Boot, JWT Authentication, and MySQL.

InterviewAce helps students prepare for technical interviews through MCQ assessments, performance analytics, leaderboard rankings, profile management, and resume management.

---

## Features

### Authentication

* Secure Login System
* JWT Based Authentication
* Protected Routes

### Dashboard

* User Dashboard
* Performance Overview
* Dashboard Analytics

### MCQ Assessment

* MCQ Test Engine
* Test Attempt Tracking
* Score Calculation

### Analytics & Rankings

* Result Analysis
* Performance Insights
* Leaderboard Rankings

### Profile Management

* Profile Management
* Education Management
* Experience Management
* Skills Management
* Projects Management
* Resume Management

### Administration

* Admin Dashboard

## Installation

### Clone Repository

```bash
git clone https://github.com/maheshtitare/InterviewAce.git
cd InterviewAce
```

### Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

Backend:

```text
http://localhost:8080
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend:

```text
http://localhost:3000
```
